### Information panel
* tile layout  (position in the grid) -  json , w -column span (12 max) , h - row span (12 max), x,y -  column ,row offset, example : ```{"w":1,"h":1,"x":3,"y":5}```
  
* tile props -  key-value dictionary of properties: 
	* "icon":str - from https://www.primefaces.org/primevue/icons,https://fontawesome.com/search?q=open&o=r
	* "icon_visibility": bool,
	* "legend":bool,
	* "title_visibility" :bool,
	* "background": css background property
	* "mask": hex color above background
